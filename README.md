# docker-centos7-nginx
A Dockerfile that installs latest CentOS 7 and NGINX

## Installation

```
git clone https://github.com/lavanco/docker-centos7-nginx.git
cd docker-centos7-nginx
docker build -t lavanco/centos7-nginx .
```
